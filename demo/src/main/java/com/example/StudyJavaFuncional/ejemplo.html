Calculadora Java Swing
Componentes Principales para Java Swing:
• JFrame: Ventana principal (300x400px aprox)
• JTextField: Display para mostrar números y resultados
• JButton (20 botones): Números (0-9), operadores (+, -, ×, ÷), punto decimal, igual, clear, backspace
• BorderLayout: Layout principal
• GridLayout: Para organizar los botones (5 filas × 4 columnas)
• JPanel: Contenedor para los botones
Tips para implementar:
• Usa GridLayout(5, 4) para los botones
• JTextField con setEditable(false) para el display
• ActionListener para manejar eventos de botones
• Considera usar Font("Arial", Font.BOLD, 16) para los botones
• El botón "0" y "=" necesitan configuración especial de tamaño
